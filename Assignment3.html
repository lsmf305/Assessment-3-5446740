Name Web Attacks
SQL Injections



Source: Medium.com 		- SQL stands for Structured Query Language, and is an attack that often occurs on a database driven website
- Malicious code is injected into vulnerable website search box
- The result allows the hacker to view, edit, and delete tables in the database

Source: Simpli Learn
Malware Attack



Source: Forgenix.com 		- is one of the most common type of cyberattacks
- refers to malicious software such as worms, spyware, ransomware, adware and trojans
- breaches take place through a network vulnerability

Source: Simpli Learn
Phishing Attack



Source: Bankinfosecurity.com 		- it is a soicial engineering attack done via impersonation of a trusted person
- it can often be done via email, and when a link or attachment is clicked, malware can be installed
- phishing can be prevented by scrutinizing emails, using anti-phishing toolbars, and updating passwords regularly

Source: Simpli Learn

Password Attack



Source: Logicxconsulting.com
- form of attack where a hacker cracks your password
- uses tools such as Aircrack, Cain, Abel, John the Ripper etc
- using strong alpha numeric passwords can help prevent password attacks

Source: Simpli Learn

Man in the middle attack



Source: Techtarget.com
- aka an evesdropping attack
- attacker gets in the way of a two party communication and hijacks the session
- MITM attacks can be prevented by being mindful of website security, using encryption, and refraining from using public wifi

Source: Simpli Learn

Denial of Service Attack



Source: f-secure.com
- Denial of Service attacks are caused by attackers flooding the systems, servers and networks of a company to exhaust the resources and bandwidth
- It prevents actual issues from being addressed
- aka DDOS when multiple attackers use compromised systems to launch this attack Source: Simpli Learn

Cryptojacking



Source: Globalsign.com
- takes place when attackers access someone elses computer for mining cryptocurrency
- access is gained by infecting a website or manipulating the victim to click on a malicious link
- Victims are unaware of this as the Crypto mining code works in the background; a delay in the execution is the only sign they might witness

Source: Simpli Learn

Zero Day Exploit



Source: wallarm.com
- Zero-Day Exploit happens after the announcement of a network vulnerability and there is no solution for the vulnerability in most cases
- hackers make sure to exploit the vulnerability even before a patch or solution is implemented for it
- can be prevented by having an incident response plan to help you deal with a cyberattack

Source: Simpli Learn

Watering Hole Attack



Source: Cybersecurityglossary.com
- a specific group is targeted by hackers on targeted websites that the group often frequent
- attackers infect these sites with malware aiming to infect the victims systems
- the hacker attempts to gain access to personal information

Source: Simpli Learn

Insider Threat



Source: Linfordco.com
- insider threats are rampant in small businesses
- reasons for this attack can include greed, malice and carelessness
- good security culture and training can help prevent this

Source: Simpli Learn
